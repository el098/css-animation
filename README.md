# CSS Animation Challenge

A small project showcasing CSS animations, hover effects, and responsive design.

## Features
- Fade‑in page transition  
- Hover animations (grow, rotate, glow)  
- Bouncing button  
- Responsive layout for mobile, tablet, and desktop  

## Tech Used
HTML • CSS • Media Queries

## Purpose
Practice CSS keyframes, transitions, and responsive design.

## How to View
Open `index.html` in your browser.
